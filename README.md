# my-admin-vue2

使用 Vue@2.7 + Antdv@1.7 + Node@18.17.0 + Vuex + Webpack 技术栈

## 已实现功能

+ 登录页
+ 表单页
+ 异常页
+ 工作台
+ 国际化
+ 菜单
+ 路由
+ 404
+ 主题切换
+ 图表引入
+ 富文本编辑器引入
+ 持久化
+ Mock
+ ESLint
+ Prettier

## 安装环境

```
npm install yarn -g
yarn install
```

## 运行项目

```
yarn serve
```

## 打包项目

```
yarn build
```
